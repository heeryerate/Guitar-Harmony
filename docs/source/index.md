# Welcome to GuitarHarmony's documentation!

To
- make/analyze music
- learn music theory
- produce tabs/midi
- create scales/chords/progressions exercise 

## Installation
```bash
    pip install guitarHarmony
```

## Examples:
```python
import guitarHarmony as gt
a = gt.Note('Fb5')
p = gt.Interval('M3')
c = gt.Chord('B','m7')
```
